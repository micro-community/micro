Introducing M3O

Next Generation Cloud Platform

Powered by Micro


M3O is a next generation cloud platform powered by Micro. Explore, discover and consume free and paid public APIs as simpler programmable building blocks all on one platform for a 10x developer experience


The current generation of cloud platforms are more than a decade old. They catered to a previous world of problems and focused on providing infrastructure building blocks. M3O provides higher level primitives to help build the next generation of software.


To learn more go to m3o.com or read the announcement post here.
